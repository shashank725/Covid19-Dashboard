The project uses on HTML, CSS, JS to make a COVID-19 Monitoring tool.

API ENDPOINTS

1) This project uses https://corona-api.com/countries to gather all the information about the countries and its codes for the ones we will show data of covid-19.

2) This project uses https://corona-api.com/countries/${countryCode} to gather all the information about a particular country.
